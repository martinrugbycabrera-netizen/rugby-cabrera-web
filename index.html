Nuestros Equipos de Cantera

Fomentamos el amor por el rugby desde las edades más tempranas. Cada categoría se enfoca en el desarrollo de habilidades motoras, trabajo en equipo y, sobre todo, la diversión.

Sub 6

Introducción al juego con foco en el movimiento y la diversión. Sin contacto.

Sub 8

Iniciación al Contacto

Se introducen gradualmente las reglas básicas y el contacto controlado.

Sub 10

Desarrollo de Habilidades

Énfasis en el pase, el placaje y las formaciones de juego sencillas.

Sub 12

Aplicación de tácticas de juego más complejas y participación en ligas.

const MATCH_SHEET_URL = "https://docs.google.com/spreadsheets/d/e/2PACX-1vSKDn3GJy4hXjEEWXf6Q_S2EPo23aVn3iTPMqCgTMGsbpcfI8rd9K2T8uvGcBYfWpeEfdf7roqlXtsY/pub?gid=0&single=true&output=csv"; async function cargarProximoPartido() { const lang = (localStorage.getItem("preferredLanguage") || "es").trim(); const res = await fetch(MATCH_SHEET_URL); const text = await res.text(); // Convertir CSV → array y eliminar cabecera const rows = text .trim() .split("\n") .slice(1) // 🔥 elimina la fila de títulos .map(r => r.split(",").map(c => c.trim())); // Buscar fila por idioma const row = rows.find(r => r[0] === lang); if (!row) { console.warn("No se encontró fila para idioma:", lang, rows); return; } document.getElementById("match-section-title").textContent = row[1]; document.getElementById("match-title").textContent = row[2]; document.getElementById("match-subtitle").textContent = row[3]; document.getElementById("match-date").textContent = row[4]; document.getElementById("match-time").textContent = row[5]; document.getElementById("match-place").textContent = row[6]; } // Al cargar document.addEventListener("DOMContentLoaded", cargarProximoPartido); // Al cambiar idioma document.querySelectorAll(".lang-btn").forEach(btn => { btn.addEventListener("click", () => { setTimeout(cargarProximoPartido, 50); }); });
